# تقرير المستخدم - مشروع موقع المجوهرات

## المهمة المنجزة
تم إنشاء واجهة أمامية حديثة وفاخرة ومتجاوبة بالكامل لموقع عرض المجوهرات والتصميم المخصص.

## تفاصيل المشروع
- **نوع المشروع**: موقع صفحة واحدة (HTML/CSS/JS)
- **السمة**: علامة تجارية للمجوهرات الفاخرة
- **لوحة الألوان**: الأسود (#1a1a1a)، الأبيض (#ffffff)، الذهبي (#d4af37)، البيج (#f5f0e8)، الوردي الفاتح (#e8d5d5)

## الميزات المنفذة
1. **شريط التنقل** - رأس ثابت مع الشعار وروابط التنقل (الرئيسية، المتجر، التصميم المخصص، من نحن، اتصل بنا) والأيقونات
2. **قسم Hero** - ملء الشاشة مع صورة مجوهرات عالية الجودة والعنوان وأزرار الدعوة للعمل
3. **قسم المنتجات** - تخطيط شبكي مع 6 بطاقات منتجات تتضمن:
   - خاتم الماس (4,850 دولار)
   - عقد ذهبي (3,200 دولار)
   - أقراط الماس (2,950 دولار)
   - سوار الماس (5,600 دولار)
   - خاتم الزمرد (3,800 دولار)
   - أقراط اللؤلؤ (890 دولار)
4. **قسم التصميم المخصص** - قسم داكن يعرض خدمات المجوهرات المصممة حسب الطلب
5. **قسم من نحن** - تراث الشركة مع شبكة الميزات
6. **قسم اتصل بنا** - نموذج اتصال مع حقول الاسم والبريد الإلكتروني والهاتف والرسالة
7. **التذييل** - أيقونات وسائل التواصل الاجتماعي والروابط السريعة وخدمة العملاء ومعلومات الاتصال

# تحديث: إنشاء الواجهة الخلفية (Backend) - نظام التجارة الإلكترونية والتصميم بالذكاء الاصطناعي

## المهمة المنجزة
تم إنشاء واجهة خلفية متكاملة باستخدام FastAPI لموقع المجوهرات مع دعم الذكاء الاصطناعي لتصميم المجوهرات.

## التقنيات المستخدمة
- **Framework**: FastAPI (Python)
- **قاعدة البيانات**: MySQL (XAMPP)
- **ORM**: SQLAlchemy مع Pydantic
- **المصادقة**: JWT Tokens
- **الذكاء الاصطناعي**: Google Gemini API

## هيكل المشروع
```
backend/
├── main.py              # نقطة الدخول للتطبيق
├── config.py            # إعدادات التطبيق
├── database.py          # اتصال قاعدة البيانات
├── auth.py              # أدوات المصادقة
├── seeder.py            # تهيئة البيانات التجريبية
├── models/              # نماذج قاعدة البيانات
├── schemas/             # مخططات Pydantic
├── routers/             # مسارات API
│   ├── auth.py          # المصادقة
│   ├── products.py      # المنتجات والفئات
│   ├── cart.py          # سلة التسوق
│   ├── orders.py        # الطلبات
│   ├── admin.py         # لوحة التحكم
│   └── ai.py            # الذكاء الاصطناعي
└── static/              # الملفات الثابتة
```

## جداول قاعدة البيانات
1. **Users** - المستخدمين

5. **Products** - المنتجات

8. **Orders & Order_Items** - الطلبات
9. **User_Generated_Designs** - التصاميم المولدة بالذكاء الاصطناعي
10. **Design_Requests** - طلبات التصميم المخصص

## نقاط API الرئيسية

### المصادقة `/api/auth`
- `POST /register` - تسجيل مستخدم جديد
- `POST /login` - تسجيل الدخول والحصول على رمز JWT
- `GET /me` - معلومات المستخدم الحالي

### المنتجات `/api/products`
- `GET /` - جلب المنتجات مع الفلاتر
- `GET /{id}` - تفاصيل منتج
- `POST /` - إضافة منتج (Admin)
- الفئات: `GET/POST /categories`

### سلة التسوق `/api/cart`
- `GET /` - عرض السلة
- `POST /items` - إضافة منتج
- `DELETE /items/{id}` - حذف منتج

### الطلبات `/api/orders`
- `GET /` - عرض الطلبات
- `POST /` - إنشاء طلب جديد
- `POST /{id}/upload-receipt` - رفع إيصال الدفع

### الذكاء الاصطناعي `/api/ai`
- `POST /generate-design` - توليد تصميم مجوهرات بالذكاء الاصطناعي
- `POST /design-requests` - إرسال طلب تصميم لصائغ

## ميزة توليد التصاميم بالذكاء الاصطناعي
```
POST /api/ai/generate-design
{
    "type": "Ring",           // النوع: خاتم، عقد، سوار
    "color": "Gold",          // اللون
    "shape": "Round",         // الشكل
    "material": "Gold",       // المادة
    "karat": "18k",          // العيار
    "gemstone_type": "Ruby",  // نوع الحجر الكريم
    "gemstone_color": "Red"   // لون الحجر
}
```

## كيفية التشغيل

### 1. إعداد قاعدة البيانات
1. تشغيل XAMPP وتفعيل MySQL
2. إنشاء قاعدة بيانات باسم `jewelry_db`

### 2. تثبيت المتطلبات
```bash
cd backend
pip install -r requirements.txt
```

### 3. إعداد ملف .env
```env
DATABASE_URL=mysql+pymysql://root:@localhost:3306/jewelry_db
GEMINI_API_KEY=your_api_key_here
```

### 4. تهيئة البيانات التجريبية
```bash
python seeder.py
```

### 5. تشغيل الخادم
```bash
uvicorn main:app --reload --port 8000
```
